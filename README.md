## 半包和粘包的处理方法

### 第一种

固定长度

### 第二种

固定分隔符

### 第三种

使用回车分隔符

### 第四种

- lengthFieldOffset=0：开始的2个字节就是长度域，所以不需要长度域偏移。
- lengthFieldLength=2：长度域2个字节。
- lengthAdjustment=0：数据长度修正为0，因为长度域只包含数据的长度，所以不需要修正。
- initialBytesToStrip=0：发送和接收的数据完全一致，所以不需要跳过任何字节。

这个偏移是发送到接受方的
